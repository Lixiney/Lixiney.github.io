STL库（二）--Pair[x,y]

Pair[x,y]

可以理解为（x,y）数学中的坐标表示
小技巧：使用typedef定义 typedef pair<int,int> PII

头文件

#include<utility>

初始化

// 两种方法初始化
pair<string,int> p("hello",1);
p = make_pair("hello",1);

first() 和 second()

p("hello",1);
p.first; //第一个元素 = hello;
p.second; // 第二个元素 = 1;

嵌套（套娃）

vector<vector<pair<int,int>>>  //与vector结合再写个vector结合即可
//套娃操作 用pair 存储3个数据
pair<int,pair<int,int>> p(1,{2,3});
