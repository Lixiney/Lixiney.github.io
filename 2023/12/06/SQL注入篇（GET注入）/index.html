SQL注入的一般流程

  • 是否存在注入并且判断注入类型
  • 判断字段数 order by
  • 确定回显点 union select 1,2
  • 查询数据库信息 @@version @@datadir
  • 查询用户名，数据库名 user() database()
  • 文件读取 union select 1,load_file(‘C:\\windows\\win.ini’)
  • 写入webshell select …into outfile…
