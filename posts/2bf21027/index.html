创建新项目

在当前目录下创建一个新的django项目，包括最基本的目录

  • 项目名称 /: 项目主目录
    • __init__ .py
    • settings.py 配置文件
    • urls.py : URL路由配置
    • wsgi.py ：WSGI 应用入口

创建新应用

创建一个新的 django 应用，包含：

  • migrations/：数据库迁移文件目录
  • _init_.py
  • admin.py：管理后台配置
  • apps.py：应用配置
  • models.py：数据库模型定义
  • tests.py：测试代码
  • views.py：试图函数

检查项目配置

这个命令会检查 django 项目是否有配置错误，包括：

  • 模型定义是否正确
  • url 配置是否有效
  • 模板设置是否正确
  • 静态文件配置等

django-admin 常用命令

命令 描述
startproject 创建一个新的django项目
startapp 创建一个新的django应用
runserver 启动开发服务器
makemigrations 生成数据库迁移文件
migrate 执行数据库迁移
shell 启动 django 交互式shell
collectstatic 收集静态文件（用于生产环境）
test 运行单元测试
