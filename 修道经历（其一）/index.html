修道经历（其一）

前言：

​ 简单记录一下拜师修行以来遇到的感受，包括眼前所见，身体所感，心中所想。信之则有，不信则无，文中涉及未经科学解释的现象，请勿盲目迷信，文中所述练功步骤，请在专业人士的指导下练习，若出现偏差走火入魔，概不负责。

​ 由于是随笔类型的文章，想到一部分则记录一部分，所以会不定时更新，内容也可能发生删改，因为练功每段时间可能会有新的感悟，或者是修改前文中所述有误的部分。

​ 学而不思则罔，思而不学则殆。要有科学的精神，真理一定是经过猜想验证假设得来的客观事实，所以遇到未知的东西，你可以去研究，但是不能一棍子打死，可以接受反驳，但要是不分青红皂白的谩骂诋毁，我且当你是个看不懂人话的禽兽。

《清静经》中所述

老君曰：大道无形，生育天地；大道无情，运行日月；大道无名，长养万物；吾不知其名，强名曰道。

这句话定义了一个新的类相当于

class 道{
	形状： 没有形状；但是生育了天地；
	感情： 没有感情；但是是日月按照一定的规律运行；
	名字： 没有名字；但是生育了万物；
}

但是，这跟修炼有什么关系呢？先看看《太乙金华宗旨》中吕洞宾吕祖爷说的

自然曰道，道无名相，一性而已，一元神而已。

道就是自然，自然就是道。后面两句话不是 A+B 的关系，更像是写的一种强调语气，元神是先天真性。

元神真性是超越时空存在的，虽然人的精气随天地的衰败而衰败，但是元神依然存在，这就是 “无极”。无极能创生天地，修炼的人入伏哦能守得住元神，就能超越阴阳、三界的束缚。

众所周知，人是阴阳调和的生物，一旦人的阴或者阳失调，就会生病，但是元神并不在其中，元神是阴阳的鼻祖。

师父说：元神的灵力及其强大，具有创造有极物的灵力和能量，可以进出任何宇宙空间和有极物体，可以控制任何宇宙空间和有极物体的运动，所以任何宇宙空间和有极物的玄机能量都不能破坏元神本体，都不能损害元神。

识神是在身体大脑中形成的可以控制生命体行为的意识体，也就是 “我”，生命从诞生的那一刻开始就不断用看听闻触等各种感官接受天地的各种信息。

比如昨天吃的什么，做了什么事，这些集合起来的意识体，就是识神。

元神和识神两者的关系，我感觉像是 root 账户和 admin 账户，admin固然能完成很多工作，但是权限最高的是 root !! 但是唯一不同的区别便是，识神的强大（也就是不断地记忆各种信息），元神便会消散。也就是说越聪明的人，他的元神越弱小。

当元神的能量不足的时候，人的精神状态便会出现问题，很多聪明绝顶的科学家也许就是这种情况，具体可以去调查取证，正如开头所写，先思考，调查取证，有例外的就不是真理。

道家讲究性命双修，命则是外在的，注重形体的修炼，性便是修炼自己的元神。

当人静坐时，识神退位，元神便会显现。

静坐不是说坐着就好，而是心无杂念地静坐，不生起任何一个念头。也就是封闭所有的意识。

当你的视觉、听觉、嗅觉、触觉、味觉，全部封闭之后，识神就不工作了，这就是无极的状态，没有思虑、恍恍惚惚，这一状态乃元神所现。

这就是“明心见性” 佛家讲的 “顿悟” ！

如何修炼？

夫道者：有清有浊，有动有静；天清地浊，天动地静；男清女浊，男动女静；降本流末，而生万物。清者，浊之源，动者，静之基；人能常清静，天地悉皆归。

所谓 “人能常清静”，若能达到清净的境界，便是天人合一，或许会发生意想不到的妙处。

元神显现之后，便会看到黑暗处生成一缕白光，这便是所谓明心见性，即看到了本来的面目，即 “真我”。

性光长什么样子？

所谓性光可以经过太阳光或者灯光激发，也就是看亮的东西之后眼前会留下痕迹，可呈现艳丽无比的各色宝珠型性光，有红色、绿色、蓝色、紫色或者其他颜色。

性光显现初期的光称之为慧光，同时伴随着幻觉、幻视等，属于下乘功夫。
